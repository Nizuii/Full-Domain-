# What is Python?

Python is a high level, interpreted programming language. Compared to other programming languagess like C, C++, Java python is more beginner friendly because first of all its syntax and readability is similar to english grammer. Compared to C and C++ which are faster at run time, Python is slower at run time, but much faster to build things.
Python has a large collection of library for almost everythingg:

- Networking
- Cryptography
- Web Scraping
- Exploit Development
- Reverse engineering helpers

## Data Types 

```bash
# Numbers
port = 443
timeout = 2.5

# Strings
ip = "192.168.1.7"
log_line = "2024-02-01 10:33:21 Failed login from 10.0.0.5"

# Boolean
is_suspicious = True

# List (ordered, mutable)
ips = ["10.0.0.1", "8.8.8.8", "192.168.1.5"]

# Tuple (ordered, immutable)
coords = ("10.0.0.1", 22)

# Dictionary (key-value, extremely important)
request = {
    "ip": "10.0.0.5",
    "status": "FAILED",
    "timestamp": "2024-02-01 10:33:21"
```
